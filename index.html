Manage browser sessions with powerful features packed into a single extension.

Capture Sessions

Save cookies, localStorage, and sessionStorage with one click. Keep your login states safe.

Instant Restore

Switch between accounts instantly. Restore any saved session with a single click.

2FA Manager

Store TOTP secrets with auto-updating codes. Generate 2FA codes directly from the extension.

Clear cookies, cache, history, and storage per tab or domain in one click.

Multiple export formats: JSON, Cookie Editor, Netscape, OWI encrypted. Import from files.

How it Works

Three simple steps to manage all your browser sessions.

1

Save Session

Click "Add Session" to capture your current login state — cookies, storage, and all.

2

Manage Sessions

View sessions by domain, edit names, backup with a master password, or delete what you don't need.

3

Restore Anytime

Click any session to instantly restore it. Switch accounts without re-entering credentials.

Get Started

Free and open source. Install in under a minute.

Installation
1Clone or download the repository from GitHub
2Run npm install and npm run build
3Open chrome://extensions and enable Developer mode
4Click "Load unpacked" and select the dist folder
// That's it. You're ready to go.
