Powerful Features

Everything you need to manage browser sessions efficiently.

Capture Sessions

Save cookies, localStorage, and sessionStorage with one click. Keep your login states safe.

Instant Restore

Switch between accounts instantly. Restore any saved session with a single click.

Organized by Domain

Sessions are automatically grouped by website. Easy to find and manage.

Encrypted Backup

Export sessions as encrypted OWI files. Your data stays secure with password protection.

Clean Tab Data

Clear cookies, cache, and storage for the current tab. Start fresh when needed.

Import & Export

Import from JSON or Cookie Editor format. Export for backup or sharing.

How it Works

Simple three-step workflow to manage your sessions.

1

Save Session

Click "Add Session" to capture your current login state including all cookies and storage data.

2

Manage Sessions

View all saved sessions organized by domain. Edit names, backup, or delete as needed.

3

Restore Anytime

Click on any session to restore it instantly. Switch accounts without re-logging in.

Get SesWi

Free and open source. Install from GitHub or load as unpacked extension.

Installation
1

Download or clone the repository from GitHub

2

Run npm install and npm run build

3

Open chrome://extensions and enable Developer mode

4

Click "Load unpacked" and select the dist folder
