# SesWi - The Advanced Cookie and Session Manager!
a Chrome Extension that simply manage cookie &amp; session securely

## Features:
- Save in JSON or Encrypted Save!
- Work Locally and doesnt require internet connection (100% ON YOUR MACHINE!)
- Save Multiple Session with current and Grouped tab for easy management.
- Add a fallback when detected website doesnt support it*
